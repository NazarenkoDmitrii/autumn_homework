# RMQ — минимумы на отрезках с обновлениями

Дан массив целых чисел. Нужно обрабатывать запросы двух типов:

- `1 l r` — вывести минимум на отрезке `a[l..r]` (1-индексация, `l ≤ r`)
- `2 i x` — присвоить `a[i] = x` (точечное обновление)

Требуется эффективное решение по времени и памяти.  
Для этой версии (с обновлениями) используйте дерево отрезков: построение `O(N)`, каждый запрос и обновление `O(log N)`.

## Формат ввода

- Первая строка: `N Q`
- Вторая строка: `N` целых чисел
- Далее `Q` строк:
  - `1 l r`
  - `2 i x`

Ограничения (типичные):
- `1 ≤ N, Q ≤ 2e5`
- значения массива и `x` помещаются в 32-битный signed int

## Формат вывода

Для каждого запроса типа `1 l r` выведите одно число — минимум на указанном отрезке.

## Пример

Ввод:
5 6
5 3 7 3 2
1 2 5
2 3 1
1 1 3
1 3 3
2 5 10
1 4 5


Вывод:


2
1
1
3

# Алгоритм Джонсона

Реализуйте алгоритм Джонсона для поиска кратчайших путей между всеми парами вершин
в ориентированном взвешенном графе. В графе могут быть отрицательные веса рёбер, петли допустимы.

Нумерация вершин ведётся с 0.

Если в графе существует отрицательный цикл, то корректные кратчайшие пути не определены.

## Формат ввода

Первая строка: `n m` — количество вершин и рёбер.

(1 ≤ n ≤ 2000, 0 ≤ m ≤ 20000)

Следующие `m` строк: `a b w` — ориентированное ребро `a -> b` веса `w`.

(0 ≤ a, b < n, -10^9 ≤ w ≤ 10^9)

## Формат вывода

Если существует отрицательный цикл, выведите:
-1

Иначе выведите `n` строк, в каждой строке `n` значений:
- кратчайшее расстояние из вершины `i` в вершину `j`,
- либо `INF`, если пути не существует.

Значения в строке разделяются пробелами.

## Пример

Ввод:
3 3
0 1 4
1 2 -2
0 2 5

Вывод:
0 4 2
INF 0 -2
INF INF 0

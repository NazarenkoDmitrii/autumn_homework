# Алгоритм Дейкстры

Реализуйте алгоритм Дейкстры для поиска кратчайших путей от одной вершины до всех остальных
в ориентированном взвешенном графе.

Нумерация вершин ведётся с 0. Петли допустимы. Кратных рёбер может быть несколько.
Вес ребра неотрицательный.

## Формат ввода

Поддерживаются два варианта:

### Вариант A
Первая строка: `n m` — число вершин и рёбер. Источник считается `s = 0`.

### Вариант B
Первая строка: `n m s` — число вершин и рёбер и источник `s`.

Далее идут `m` строк вида `u v w` — ребро `u -> v` веса `w`.

Ограничения (типичные для Дейкстры):
- 1 ≤ n ≤ 10^5
- 0 ≤ m ≤ 10^5
- 0 ≤ u, v < n
- 0 ≤ w ≤ 10^9

## Формат вывода

Выведите одну строку из `n` значений:
- расстояние от `s` до вершины `i`,
- либо `INF`, если вершина недостижима.

Значения разделяются пробелами.

## Пример

Ввод:
4 3 0
0 1 1
1 2 2
2 3 3


Вывод:
0 1 3 6

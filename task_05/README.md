# Минимальное остовное дерево с ограничением на степень

Дан связный неориентированный взвешенный граф. Нужно построить остовное дерево минимального веса
с ограничением на степени вершин: степень каждой вершины в дереве не должна превышать `d`.

Петли и кратные рёбра возможны.

## Формат ввода

Первая строка: `n m d`
- `n` — число вершин (1 ≤ n ≤ 1000)
- `m` — число рёбер (1 ≤ m ≤ 10000)
- `d` — максимальная степень вершины в остовном дереве (1 ≤ d ≤ n - 1)

Далее `m` строк: `u v w`
- `1 ≤ u, v ≤ n`
- `1 ≤ w ≤ 1000`
- ребро неориентированное

## Формат вывода

Если существует остовное дерево, в котором степень каждой вершины ≤ `d`, выведите его суммарный вес.

Иначе выведите строку:
`Невозможно построить остовное дерево с заданным ограничением степени`

## Замечание

Решение строит дерево жадным вариантом алгоритма Краскала с проверкой ограничения на степени.
